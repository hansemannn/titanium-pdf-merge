# Titanium iOS PDF Merge

Merges a given number of PDF files into one file using the `PDFKit` framework

## Requirements

- [x] iOS 11+
- [x] Titanium SDK 9+

## API's

### Methods

- `mergedPDF(paths) -> Ti.Blob`: Returns a `Ti.Blob` that represents a merged PDF from the given PDF files
